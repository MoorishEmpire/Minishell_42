═══════════════════════════════════════════════════════════════════════════════
                              KEY CHANGES
═══════════════════════════════════════════════════════════════════════════════

1. COMMAND STRUCTURE (t_cmd):
   OLD: cmd->cmd, cmd->args, cmd->file (linked list)
   NEW: cmd->argv, cmd->redirect[], cmd->file[] (arrays)

2. REDIRECTION HANDLING:
   OLD: Iterate through t_file linked list
   NEW: Parallel arrays - redirect[i] matches file[i]

3. COMMAND EXECUTION:
   OLD: Check cmd->cmd for builtin
   NEW: Check cmd->argv[0] for builtin

4. MAIN FLOW:
   Input → Tokenize → Expand → Wildcard → Strip → Parse → Execute
